Variables in Attribute Table

sourceid: Identificacion number of the road segment

roadname: Name of the road

functional: Type of Road

Vm: Traffic Volume [No. veh/h]

Nth: Number of Lanes

Sr: Motorized Vehicle Running Speed [km/h]

Wv: Width of lane, width of bike lane and width of shoulder [m]

W1: Width of bike lane and width of shoulder [m]

Ppk: On-Street Parking proportion

Wbuff: Width of the Buffer [m]

WaA: Width of the Sidewalk [m]

PC: Pedestrian Crossing

POI: Points of Interest

PT: Public Transports Stops

PS: Public Services

CR: Curb Ramps count

L: Lighting count

PLOS formula in QGIS

6.0468+(0.0091*"Vm"/(4*"Nth"))+4*(("Sr"*0.621371/100) ^ 2)-1.2276* ln( "Wv"*3.28084+0.5*"W1"*3.28084+50*"Ppk"+"Wbuff"*3.28084+"WaA"*3.28084* if("WaA"*3.28084>10,6-0.3*10,6-0.3*("WaA"*3.28084) ))
